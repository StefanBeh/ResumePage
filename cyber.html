Cyber Security

Durch mein Interesse an der Web-Entwicklung bin ich auf die Frage gestoßen, wie ich meine Web-Apps sicher gestalten kann. Dabei bin ich auf die Seite "tryhackme.com" gestoßen. Diese und "hackthebox.com" sind die zwei größten Anbieter in Bezug auf Cybersecurity Schulungen.

Try hack me bietet sehr gut geführte Schulungen mit virtuellen Maschinen, die für die jeweilige Übung gestartet werden. Es werden sowohl Inhalte für Windows, als auch für Linux geschult.

Es gibt 3 verschiedene Lernpfade.

  • SOC (Security Operations) Spezialist
  • Penetrationtester
  • Security Engineer

SOC Spezialist

Der SOC Spezialisten Pfad befasst sich mit der Entdeckung, Analyse und Abwehr von Cyberangriffen.
Folgende Tools werden behandelt:

  • Snort (Intrusion Detection und Intrusion Prevention)
  • Zeek (Intrusion Detection)
  • Brim (Intrusion Detection)
  • Yara (Detection Rule Language)
  • NetworkMiner (Netzwerk-Analyse)
  • Wireshark (Netzwerk-Analyse)
  • TShark (Netzwerk-Analyse auf CLI Basis)
  • Elastic Search und Kibana (Log-Analyse)
  • Splunk (Log Analyse)
  • Wazuh (Intrusion Detection, Log-Management, Compliance-Überwachung und Vulnerability Detection)
  • Velociraptor (Endpunktüberwachung)
  • Autopsy (Datenträger-Analyse und Wiederherstellung)
  • Volatility (RAM-Analyse)
  • Sysmon (Endpunkt Logging)
  • Osquery (Endpunkt Log Query Tool)
Außerdem werden Security Frameworks geschult:
  • MITRE
  • Pyramid of Pain
  • Cyber Kill Chain
  • Unified Kill Chain
  • Diamond Model

Pentester

Der Pentester Pfad befasst sich mit der Durchführung von authorisierten Eindringungsversuchen in ein bestehendes System. Dabei wird getestet ob ein eindringen möglich ist, und wenn, ob es entdeckt und verhindert werden kann.
Folgende Tools werden behandelt:

  • Webseiten Analyse
  • Authentication Bypass
  • SSRF (Server Side Request Forgery)
  • XSS (Cross-Site Scripting)
  • Command Injection
  • SQL Injection
  • Burp Suite (Webanfragen abfangen und verändern)
  • Nmap (Port Scanning)
  • Metasploit (Automatisiertes Tool für Eindringungsversuche)
  • Moniker Link
  • Linux Privilege Escalation
  • Windows Privilege Escalation
Außerdem werden Grundlagen geschult:
  • MITRE
  • OWASP (Top 10 Verwundbarkeiten von Webseiten)
  • DNS, HTTP, SSH, UDP, SMTP, IMAP Protokolle
  • Active Directory
  • Linux Shell
  • Powershell
  • Cryptography
  • Java Script
  • SQL
  • Firewall
  • IDS (Intrusion Detection Systeme)
  • Hashing
